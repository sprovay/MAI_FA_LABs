=== ТЕСТОВЫЙ ВЫВОД OVERFPRINTF ===

1. Римские цифры:
   1999 = MCMXCIX
   2024 = MMXXIV
   49   = XLIX

2. Цекендорфово представление:
   0  = 01
   1  = 11
   4  = 1011
   13 = 0000011

3. Системы счисления:
   255 в 16-чной (нижний): ff
   255 в 16-чной (верхний): FF
   -42 в 10-чной: -42

4. Преобразование из строки:
   "ff" из 16-чной = 255
   "1010" из 2-чной = 10
   "ABC" из 16-чной = 2748

5. Дамп памяти:
   int 42: 00101010 00000000 00000000 00000000
   unsigned int 123: 01111011 00000000 00000000 00000000
   float 3.14: 11000011 11110101 01001000 01000000
   double 123.456: 01110111 10111110 10011111 00011010 00101111 11011101 01011110 01000000

6. Комбинированные тесты:
   Число 20: римское=XX, цекендорф=0101011
   Число 255: в 2-чной=11111111, в 16-чной=FF
